1. PTP status & 1.0 release plans

Two components have been released, the remained of PTP is currently at RC2. Status of components as follows:

  • FDT - v1.0 released
  • MPI development tools - v1.0 released
  • Runtime - cleanup, bug fixing, testing
  • Debugger - functionality complete, testing

The licensing issue that was discovered with the debugger has now been addressed.

We are still working on packaging the runtime and debugger components.

2. Other Project Status

IBM

IBM have firmed up their project plans for PTP:

  1. OpenMP tools, similar to those provided for MPI. Currently working on extending the CDT parser to support pragmas. Changes to the parser will need to be incorporated back into the CDT code.
  2. Looking at integration with HPC Toolkit, support for Blue Gene, tracing/visualization, etc.
  3. Demonstrate more capabilities of PTP. Currently only demonstrating MPI tools, but would like to be able to demonstrate runtime and debugger functionality.

Beth asked about coding conventions for their development work. Eclipse coding conventions can be found here.

U of Oregon

Currently working on integrating TAU with the PTP launcher.

Cray

Interested in finding appropriate places for collaboration. One area of possibility is with the debugger.
