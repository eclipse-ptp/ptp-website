Fortran Development Tools

Use the following URL in the update manager:

http://update.eclipse.org/technology/ptp/releases

To access the Update Manager, select Software Updates->Find and Install from the Help menu. Select Search for new features to install and click on Next. Choose New Remote Site, and in the New Update Site dialog, enter a name for the site (any name) and enter the above URL for the update site. Select the update site(s) to include in the search and click on Finish. You can then expand the site in the Search Results view to reveal the available downloads.

Currently, only a pre-release version of FDT 0.9-31M5a is available.

FDT 0.9-31M5a Note: This will only work with Eclipse 3.1-M5a and no other version. Only Linux and Mac OS X are supported. Only IBM XLF Fortran is supported, though you can change the build properties to try out other Fortran compilers.
