Installing Eclipse and PTP

Updated a bit for PTP Release 1.1.1 (1 November 2007): PTP update site now available to install PTP.

Note that PTP Release 1.1.1 requires Eclipse 3.2.2 and CDT 3.1.2. PTP 2.0, available around early 2008, will use the more recent Eclipse 3.3 (Europa) and CDT 4.0.

To get Eclipse 3.2,

  • Go to eclipse Downloads, "Find out more..." under Eclipse Classic, and click on Archived Platform Builds. Then click on Build 3.2.2 and download the SDK for your platform.

To install Eclipse, the CDT, and then PTP, do the following:

  1. Download either the "light" (platform) version of Eclipse or the full "SDK" version: go to the Eclipse download site for release 3.2.2 as described above, and then do one of the following:
    • For the "light" version, scroll down to the heading "Platform Runtime Binary" and download the tar (or zip) file for your platform. This is about 1/3 to 1/4 the size of the full SDK, and doesn't include the Java or Eclipse Plug-in development parts, which I assume you don't need. The UI is simpler since you don't have all those additional options.
    • The Eclipse SDK works just fine, though, if you prefer that, and may be more complete. On the main downloads page for 3.2.2 above, see "Eclipse SDK" near the top of the page, and download the SDK for your platform.

      Eclipse 3.3 has a pre-built version for C/C++ development, so that should simplify installation in the future.

    After downloading and extracting Eclipse (zip or tar file), there's nothing else to the installation process. In the extracted folder, simply run the eclipse executable file. It will ask for a workspace location; the default is fine.

  2. Install the CDT (C/C++ Development Toolkit) as well. You can do this from the Eclipse UI as follows:
    1. Help > Software Updates > Find and Install...

    2. Select "Search for new features to install",

    3. Click Next, Select "Callisto Discovery Site" and click "Finish."
      (The Callisto discovery site lists Eclipse plug-ins guaranteed to work with the 3.2 version of Eclipse.)

    4. When the Callisto Update site is loaded you see another Dialog.
      Expand the "Callisto Discovery Site" item and check the "C and C++ Development" item.

    5. Click Next, accept the license terms, Click Next, and click Finish.
    6. On the Verification page, click "Install All."
    7. When it's done, click "Yes" to restart the Eclipse workbench. Then you can install PTP from its update site as well. (Jump to step 3. below...) Or click "no" to close down Eclipse if you prefer to install PTP from downloaded tar.gz files.

    8. Note: the above installs the runtime version of CDT. If you intend to develop plug-ins with CDT, be sure to get the full SDK version.
      The CDT SDK as well as the runtime are available from the CDT Update Site (as opposed to the Callisto discovery site mentioned above). The CDT Update Site is available by entering the following URL as the Remote Site in the Update Manager.
      • http://download.eclipse.org/tools/cdt/releases/callisto

  3. To download and install PTP, either
    • (Recommended) Use the update manager and the PTP update site. See the update site for more details.
    • or close Eclipse, download PTP and PLDT from the PTP downloads page.

    • Now follow the instructions in the release notes and installation instructions page. (You'll need to build a small stub for your platform.)
