ptp work tasks
parallel tools platform subproject

The following is a list of development tasks for the parallel tools platform. If you are looking for areas to contribute to ptp, then this is a good place to start. Simply pick one of these tasks and start working on it. If there are new features or enhancements that you would like to see added to this list, please open a bugzilla request.
